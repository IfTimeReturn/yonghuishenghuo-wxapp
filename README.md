# yonghuishenghuo-wxapp
### 项目说明：
仿YH永辉生活+小程序。

本项目只涉及到前端内容，所用数据全部都是假数据。

### 目录结构：

- components —— 自定义组件
- images —— 项目页面相关图片
- libs —— 项目所用到的一些数据
- pages —— 项目页面文件
- style —— 公共wxss文件
- template —— 模版文件
- utils —— 公共js文件
